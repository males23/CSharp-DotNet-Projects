# CSharp-DotNet-Projects
Projets d'entraînement en C# et .NET

Exécution des programmes C#. Il s’agit des programmes simples qui impriment le message « Hello World ! », par exemple. Ces programmes utilisent la Console.WriteLine méthode pour imprimer les messages. Console est un type qui représente la fenêtre de console. WriteLine est une méthode du type Console qui imprime une ligne de texte dans cette console de texte. Et unevolution avec les bases C#.
