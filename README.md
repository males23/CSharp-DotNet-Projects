# CSharp-DotNet-Projects
Projets d'entraînement en C# et .NET

Nous venons d’exécuter notre premier programme C#. Il s’agit d’un programme simple qui imprime le message « Hello World ! » Elle utilise la Console.WriteLine méthode pour imprimer ce message. Console est un type qui représente la fenêtre de console. WriteLine est une méthode du type Console qui imprime une ligne de texte dans cette console de texte.
